Este repositório centraliza minha evolução diária na programação, registrando de forma contínua todos os exercícios, códigos e desafios que desenvolvo.

## 🎓 Formação & Contexto

* **Instituição:** Aluno de Programação Web na Uninter.
* **Foco Técnico:** Construção de uma base lógica inabalável e domínio pleno no desenvolvimento de software focado no mercado internacional até 2027.

## 📱 Meu Setup

* **Ambiente:** iOS (iPhone) via Spck Editor + Teclado Físico.

## 📂 Organização do Repositório

* `/web-html5-css3` -> Módulos, exercícios e desafios práticos de Desenvolvimento Web (Curso em Vídeo / Grade Curricular). Dominados 10 exercícios e 2 desafios com tags semânticas e atributos seguros.
* `/fundamentos-cpp` -> Códigos, algoritmos e lógicas de fixação estruturadas em C++ (eXcript). Domínio de escopos, loops sentinela (`while`) e travas contra divisão por zero.
* `/desafios-python` -> Coleção massiva de 64 algoritmos e desafios práticos focados em estruturas de dados, automação e resolução avançada de problemas.
